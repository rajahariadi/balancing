Instruksi Soal

  1. Buat dulu branch anda dari branch master/main
  2. Ketentuan nama branch adalah: login/tahunangkatan_nama
  3. Lokasi project kalian beradan di folder login/tahunangkatan_nama
  4. Jangan lupa pindah ke branch nya masing masing
  5. Buat 1 halaman login dan 2 halaman home
  6. Halaman home nya adalah Halaman admin dan Halaman User
  7. Dan apabila ada sesi login, dia tidak bisa akses ke halaman login dan begitu juga sebaliknya
  8. Lakukan PR dengan nama branch tadi dan jangan lupa untuk melampirkan hasil output dari program
  9. Jangan lupa di PR nya untuk menambahkan alip dan raja sebagai reviewer, ada di bagian kanan ketika PR
