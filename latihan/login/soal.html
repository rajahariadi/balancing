Instruksi Soal

  1. Buat dulu branch anda dari branch master/main
  2. Ketentuan nama branch adalah: login/tahunangkatan_nama
  3. Lokasi project kalian beradan di folder login/tahunangkatan_nama
  4. Buat 1 halaman login dan 2 halaman home
  5. Halaman home nya adalah Halaman admin dan Halaman User
  6. Dan buat apabila login sebagai admin tidak bisa masuk ke halaman user begitu juga sebaliknya
  7. Lakukan PR dengan nama branch tadi dan jangan lupa untuk melampirkan hasil output dari program
  8. Jangan lupa di PR nya untuk menambahkan alip dan raja sebagai reviewer, ada di bagian kanan ketika PR
